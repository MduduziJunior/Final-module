Help / How to Use

  1. Register or login. Use your username and PIN. With Supabase configured, cloud login can sync your profile.
  2. Record transactions first. Add income, expenses, savings, investments, and debt payments through the Transactions page only.
  3. Attach receipt proof when available. Files upload to Supabase Storage when signed in. JSON keeps only metadata/path, not the image itself.
  4. Use search and filters. Review transactions by month, type, review status, or keyword.
  5. Check the Review & Error Center. Fix duplicates, missing receipt warnings, cloud errors, and negative balances.
  6. Use the General Ledger. Automatic postings create Bank/Cash, AR, AP, assets, liabilities, equity, income, expense, and custom account views.
  7. Close each month. Use the monthly checklist and export JSON before important cloud changes.
  8. Save to cloud. Manual Save to Cloud is always available; auto-save queues 30 seconds after changes when signed in.
  9. Use JSON only as backup. Export JSON before resets, GitHub changes, or manual cloud loads.
